# 9mmRIOT HP 定期更新チェックリスト

## イベントサイクルごとの更新作業

9mmRIOTのHPは、イベントの進行に合わせて以下のタイミングで更新が必要です。

---

## 1. 開催発表時（例: 4/9）

| 作業 | ファイル | 内容 |
|---|---|---|
| NEWS追加 | `news.json` | 開催決定のNEWSを先頭に追加 |
| NEXT EVENT更新 | `index.html` | 日程・会場を更新 |
| EVENT詳細更新 | `pages/event.html` | NEXT EVENTセクションの日程・会場を更新 |
| フライヤー差替 | `images/flyers/riot9-flyer.jpg` | 告知画像を上書き |
| Netlify URL変更 | Netlify管理画面 | サイト名を `9mmriot` 等に変更（初回のみ） |
| git push | ターミナル | 変更をpushしてデプロイ |

---

## 2. バンド募集開始時（例: 4/19）

| 作業 | ファイル | 内容 |
|---|---|---|
| NEWS追加 | `news.json` | 募集開始のNEWSを追加 |
| エントリーページ更新 | `pages/entry-riot9.html` | ステータスを「募集中」に変更、フォームURLのコメントを外す |
| Googleフォーム公開 | Google Forms | フォームの「回答を受付中」をON |
| git push | ターミナル | 変更をpushしてデプロイ |

---

## 3. 募集締切時

| 作業 | ファイル | 内容 |
|---|---|---|
| NEWS追加 | `news.json` | 募集締切のNEWSを追加 |
| エントリーページ更新 | `pages/entry-riot9.html` | ステータスを「締切」に変更 |
| Googleフォーム締切 | Google Forms | フォームの「回答を受付中」をOFF |
| git push | ターミナル | 変更をpushしてデプロイ |

---

## 4. 出演バンド発表時

| 作業 | ファイル | 内容 |
|---|---|---|
| NEWS追加 | `news.json` | 出演バンド発表のNEWSを追加 |
| EVENT詳細更新 | `pages/event.html` | 出演バンド情報を追加 |
| git push | ターミナル | 変更をpushしてデプロイ |

---

## 5. チケット販売開始時

| 作業 | ファイル | 内容 |
|---|---|---|
| NEWS追加 | `news.json` | チケット販売のNEWSを追加 |
| チケットボタン有効化 | `pages/event.html` | チケット予約ボタンのコメントを外してURLを設定 |
| git push | ターミナル | 変更をpushしてデプロイ |

---

## 6. イベント開催後

| 作業 | ファイル | 内容 |
|---|---|---|
| イベント詳細ページ作成 | `pages/riot9.html` | riot8.htmlをコピーして作成、出演バンド・開催情報を記載 |
| フライヤー配置 | `images/flyers/riot9-flyer.jpg` | 正式なフライヤー画像に差替 |
| YouTubeアーカイブ追加 | `pages/riot9.html` | YouTubeサムネイルリンクを追加 |
| PAST EVENTS更新 | `pages/event.html` | PAST EVENTSにRIOT9を追加 |
| トップページ更新 | `index.html` | 過去イベントカードにRIOT9を追加 |
| ABOUT更新 | `pages/about.html` | HISTORYにRIOT9を追加 |
| NEWSタグ追加 | `news.json` | アーカイブ公開のNEWSを追加 |
| MOVIE差替 | `index.html` | トップページのYouTubeサムネイルを最新に差替 |
| git push | ターミナル | 変更をpushしてデプロイ |

---

## 7. 次回イベント準備開始時

| 作業 | ファイル | 内容 |
|---|---|---|
| 募集ページ作成 | `pages/entry-riot10.html` | entry-riot9.htmlをコピーして作成 |
| 日程・会場・条件を書き換え | 新しいentry-riot10.html | 持ち時間・出演費・募集開始日等を更新 |
| event.htmlのリンク更新 | `pages/event.html` | 募集ボタンのリンク先を新しいファイルに変更 |
| NEXT EVENT更新 | `index.html` + `pages/event.html` | 次回イベント情報に差替 |
| git push | ターミナル | 変更をpushしてデプロイ |

---

## 定期チェック（イベントに関係なく）

| 頻度 | 作業 |
|---|---|
| 随時 | Xでの告知に合わせてNEWSを追加 |
| 年1回 | フッターのコピーライト年を更新（例: 2016-2027） |
| 年1回 | ABOUTの統計数字（開催回数・出演バンド数・年の歴史）を更新 |
