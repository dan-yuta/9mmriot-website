# デザイン修正依頼プロンプト

## 用途

テキスト変更、レイアウト修正、画像差替などのデザイン修正を依頼する。

## プロンプト例

### テキスト変更

```
D:\200_OneDrive\OneDrive\10_9mmRIOT関連\999_HP関連\10_ソースコード
の以下のページのテキストを修正してgit pushしてください。

ページ: pages/about.html（またはindex.html等）
修正前: 〇〇〇〇
修正後: △△△△
```

### 画像差替

```
以下の画像をHPに反映してgit pushしてください。
圧縮もお願いします。

画像ファイル: C:\Users\dangi\Downloads\新しい画像.jpg
配置先: images/flyers/riot9-flyer.jpg（上書き）
```

### レイアウト修正

```
D:\200_OneDrive\OneDrive\10_9mmRIOT関連\999_HP関連\10_ソースコード
の以下のページのレイアウトを修正してgit pushしてください。

ページ: 対象ページ
問題: （スクリーンショットを添付するか、問題を説明）
希望: （どうなってほしいか）
```

## 補足

- スクリーンショットを添付すると、具体的な修正がしやすい
- スマホ表示の問題は、ブラウザのDevToolsでモバイル表示にした状態のスクリーンショットが有効
