# Google フォーム応募システム 構築手順

フォームで応募を受付 → 既存スプレッドシートの「バンドメンバープロフィール」シートに自動反映する仕組みを作ります。

---

## 全体の流れ

```
応募者がフォーム入力
  ↓
Google フォームが回答を「フォームの回答」シートに記録
  ↓
Google Apps Script が自動で起動
  ↓
「バンドメンバープロフィール」シートにバンド名・パート・メンバー名を追記
```

---

## STEP 1: Google フォームを作成する

### 1-1. フォームを作成

1. [Google Forms](https://forms.google.com) にアクセス
2. 「空白のフォーム」を選択
3. タイトルを「9mmRIOT vol.09 出演バンド応募フォーム」に設定
4. フォームの概要（説明文）に以下を設定:

```
9mmRIOT vol.09 出演バンド応募フォームです。

■ 日程: 2026年9月26日（土）
■ 会場: 吉祥寺 Planet K

・バンドメンバーが確定した状態でご応募ください。
・主催からの返信をもってエントリー完了となります。
・6人以上のバンドは、備考欄に追加メンバーのパートと名前をご記入ください。
```

### 1-2. フォームの項目を追加

以下の順番で項目を追加してください。

| # | 項目名 | タイプ | 必須 | 備考 |

| 2 | メンバー人数 | プルダウン | 必須 | 選択肢: 2, 3, 4, 5 |
| 3 | メンバー1 パート | プルダウン | 必須 | 選択肢: Vo., Gt., Vo./Gt., Ba., Dr. |
| 4 | メンバー1 名前 | 記述式（短文） | 必須 | 説明文:「ニックネームでもOKです！」 |
| 5 | メンバー2 パート | プルダウン | 必須 | 同上の選択肢 |
| 6 | メンバー2 名前 | 記述式（短文） | 必須 | |
| 7 | メンバー3 パート | プルダウン | 任意 | |
| 8 | メンバー3 名前 | 記述式（短文） | 任意 | |
| 9 | メンバー4 パート | プルダウン | 任意 | |
| 10 | メンバー4 名前 | 記述式（短文） | 任意 | |
| 11 | メンバー5 パート | プルダウン | 任意 | |
| 12 | メンバー5 名前 | 記述式（短文） | 任意 | |

### 1-3. フォームの設定

- 「設定」タブ → 「回答」→「メールアドレスを収集する」をON（任意）
- 「プレゼンテーション」→ 確認メッセージを設定:
  「ご応募ありがとうございます！主催からの返信をもってエントリー完了となります。しばらくお待ちください。」

## STEP 2: HPにフォームのリンクを設定

募集開始時に `pages/entry-riot9.html` を編集します。

```html
<!-- この部分のコメントを外してURLを設定 -->
<a href="GOOGLE_FORM_URL" target="_blank" rel="noopener" class="btn btn-primary">
  <i class="fa-solid fa-pen-to-square"></i> エントリーフォームはこちら
</a>
```

`GOOGLE_FORM_URL` をフォームの公開URLに差し替えてください。

## STEP 3: 応募が来たらスプレッドシートに手動転記

フォームまたはX DMで応募が届いたら、スプレッドシートに**手動で**転記します。

### なぜ自動転記ではなく手動転記なのか

Google Apps Script（GAS）による自動転記も技術的には可能ですが、以下の理由から手動転記を採用しています。

| 観点 | 自動転記の問題 | 手動転記のメリット |
|---|---|---|
| スプレッドシートの構造 | テンプレート行（色分け・チェックボックス付き）と自動転記の行が一致しない | テンプレートに合わせて自由に配置できる |
| バンドごとの行数 | 4人バンドと5人バンドで行数が変わり、テンプレートとずれる | パート構成に合わせて調整できる |
| 色分け（青・赤・緑セル） | GASではセルの色分けが反映されない | 手動で色を付けられる |
| X DM経由の応募 | フォーム経由のみ対応。DMは手動転記が必要 | どちらの応募も同じ手順で対応 |
| 作業量 | 設定・保守が必要 | 1バンドあたり1〜2分程度 |

### 転記先

スプレッドシート `2026/yy/dd_9mmRIOT vol.09` の「バンドメンバープロフィール」シート

### 転記する列

| 列 | 内容 | 備考 |
|---|---|---|
| B | 出演番号 | 1人目の行のみ記入（連番） |
| C | バンド名 | 1人目の行のみ記入（青セル） |
| D | パート | Vo., Gt., Vo./Gt., Ba., Dr. |
| E | メンバー名 | バンド代表者は赤セル |
| F | バンドPR | 1人目の行のみ（任意） |

### 転記の手順

```
1. フォームの回答一覧（またはX DM）を確認
2. スプレッドシートのテンプレート行にバンド名・メンバー名を記入
3. バンド名のセルを青色に設定
4. バンド代表者の名前セルを赤色に設定
5. 応募者にメールまたはDMで受付確認 + LINEグループ招待を送る
```

---

## 運用の流れ

```
1. 募集開始
   → Xで告知 + entry-riot9.html のフォームリンクを有効化

2. 応募受付中
   → フォーム回答 or X DM を確認
   → スプレッドシートに手動転記（1バンド1〜2分）
   → 応募者にメールまたはDMで受付確認
   → LINEグループに招待
   → 主催からの返信をもってエントリー完了

3. 募集締切
   → フォームの「回答を受付中」をOFFに変更
   → entry-riot9.html のステータスを「締切」に変更
```
