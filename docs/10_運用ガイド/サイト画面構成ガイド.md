# 9mmRIOT サイト画面構成ガイド

このドキュメントでは、9mmRIOTのホームページの画面構成を説明します。
どのページに何があるか、どこを編集すれば何が変わるかがわかるようになっています。

---

## サイト全体の構成図

```
トップページ（index.html）
│
├── [NEWS]    → pages/news.html（お知らせ一覧）
├── [ABOUT]   → pages/about.html（9mmRIOTとは）
├── [EVENT]   → pages/event.html（イベント一覧）
│               ├── pages/riot8.html（RIOT8の詳細）
│               ├── pages/riot7.html（RIOT7の詳細）
│               ├── pages/riot6.html（RIOT6の詳細）
│               ├── pages/riot5.html（RIOT5の詳細）
│               ├── pages/riot4.html（vol.04の詳細）
│               ├── pages/riot3.html（vol.03の詳細）
│               ├── pages/riot2.html（vol.02の詳細）
│               └── pages/riot1.html（vol.01の詳細）
├── [MOVIE]   → トップページ内のセクション
├── [CONTACT] → トップページ内のセクション
└── [ENTRY]   → pages/entry-riot9.html（出演バンド募集）
```

## トップページ（index.html）

トップページは1ページの中にいくつかのセクションが並んでいます。
スクロールすると順番に表示されます。

### セクション一覧（上から順）

| セクション | 内容 | 詳細ページ |
|---|---|---|
| HERO | ロゴ・キャッチコピー・ボタン | — |
| NEWS | 最新のお知らせ3件 | news.html |
| ABOUT | 9mmRIOTの紹介文 | about.html |
| EVENT | 次回イベント + 過去イベント4件 | event.html |
| MOVIE | YouTubeアーカイブ（RIOT8の動画） | — |
| CONTACT | XへのDM案内 | — |

### ヘッダー（画面上部の固定バー）

すべてのページに共通で表示されます。

```
[ロゴ画像 + 9mmRIOT]  TOP  NEWS  ABOUT  EVENT  MOVIE  CONTACT  [X] [YouTube]
```

- TOP → トップページの先頭にスクロール
- NEWS / ABOUT / EVENT → それぞれの詳細ページに移動
- MOVIE / CONTACT → トップページ内のセクションにスクロール
- X / YouTube → 外部サイトが新しいタブで開く

## 各ページの説明

### NEWS 一覧（pages/news.html）

お知らせの一覧ページです。カテゴリで絞り込みができます。

- **データの管理**: `news.json` ファイルを編集するだけで更新できます
- **カテゴリ**: EVENT（赤）/ INFO（グレー）/ ENTRY（緑）/ MEDIA（紫）
- 詳しい更新方法 → [NEWS更新手順.md](NEWS更新手順.md)

### ABOUT 詳細（pages/about.html）

9mmRIOTの紹介ページです。

- 9mmRIOTとは（紹介文）
- HISTORY（年表形式の開催履歴）

### EVENT 一覧（pages/event.html）

イベント情報のハブページです。

- NEXT EVENT（次回開催の情報カード）
- PAST EVENTS（過去イベント一覧。フライヤー画像付き）
  - 各イベントをクリックすると詳細ページに移動

### イベント詳細（pages/riot1.html 〜 riot8.html）

各回のイベント詳細ページです。

- フライヤー画像（左側）
- 開催情報テーブル（日程・会場・開場時間・チケット）
- 出演バンド一覧
- YouTubeアーカイブ（RIOT5以降）
- 前後のイベントへのナビゲーション

### 出演バンド募集（pages/entry-riot9.html）

バンド募集要項のページです。

- 募集ステータス（準備中 / 募集中 / 締切）
- X での情報発信案内
- はじめての方への案内
- 応募条件・注意事項
- お問い合わせ（XのDM）

## よくある編集パターン

### NEWSを追加したい

`news.json` を開いて、先頭に1件追加するだけ。
→ 詳しくは [NEWS更新手順.md](NEWS更新手順.md)

### 次回イベントの情報を更新したい

1. `pages/event.html` を開く
2. 「NEXT EVENT」のセクション内のテーブルを書き換える
3. `index.html` の EVENT セクション内のカード情報も同様に更新

### フライヤー画像を追加・差し替えたい

1. 画像を `images/flyers/` に配置
2. ファイル名は `riot[回数]-flyer.jpg`（例: `riot9-flyer.jpg`）

### 新しいイベント詳細ページを追加したい

1. `pages/riot8.html` をコピーして `pages/riot9.html` を作成
2. 中身（日程・会場・バンド名など）を書き換える
3. `pages/event.html` のPAST EVENTS に1件追加
4. 前後のページのナビゲーションリンクを更新

### ロゴ画像を差し替えたい

1. 新しい画像を `images/logo.png` に上書き保存
2. 全ページに自動反映（ファイル名を変えなければ編集不要）
