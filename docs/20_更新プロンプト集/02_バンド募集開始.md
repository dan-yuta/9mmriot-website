# バンド募集開始プロンプト

## 用途

バンド募集を開始する（エントリーページの更新 + NEWS追加 + git push）。

## プロンプト

```
9mmRIOT vol.XX のバンド募集を開始します。
以下の対応をしてgit pushしてください。

1. pages/entry-riotXX.html のエントリーセクションを更新
   - 準備中テキストを削除
   - 以下のフォームURLのコメントを外して有効化
   - フォームURL: https://forms.google.com/XXXXXXX

2. news.json に募集開始のNEWSを追加
   - 日付: YYYY.MM.DD
   - カテゴリ: ENTRY
   - タイトル: 9mmRIOT vol.XX 出演バンド募集開始！
   - リンク先: pages/entry-riotXX.html
```

## 補足

- フォームURLはGoogleフォームの「送信」→「リンク」からコピー
- Googleフォーム側でも「回答を受付中」をONにする
