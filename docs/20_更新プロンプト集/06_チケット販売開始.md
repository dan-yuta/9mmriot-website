# チケット販売開始プロンプト

## 用途

チケット販売を開始する（ボタン有効化 + NEWS追加 + git push）。

## プロンプト

```
9mmRIOT vol.XX のチケット販売を開始します。
以下の対応をしてgit pushしてください。

1. pages/event.html のチケット予約ボタンのコメントを外して有効化
   - チケットURL: https://XXXXXXX

2. pages/event.html のチケット欄を更新
   - 「詳細は追って発表」→「¥X,XXX（+1DRINK ¥XXX）」

3. index.html のNEXT EVENTカードのチケット欄も同様に更新

4. news.json にチケット販売開始のNEWSを追加
   - 日付: YYYY.MM.DD
   - カテゴリ: EVENT
   - タイトル: 9mmRIOT vol.XX チケット予約開始！
   - リンク先: pages/event.html
```
