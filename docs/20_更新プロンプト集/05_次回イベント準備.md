# 次回イベント準備プロンプト

## 用途

次回イベントの開催発表に合わせて、NEXT EVENT情報の更新・募集ページの作成を行う。

## プロンプト

```
9mmRIOT vol.XX の開催が決定しました。以下の情報をもとにHPを更新してgit pushしてください。

【次回イベント情報】
- 日程: YYYY年M月D日（曜日）
- 会場: 会場名
- 募集開始予定: M/D（曜日）HH:MM〜

【告知画像】
（告知画像のファイルパスを指定）

【作業内容】
1. index.html のNEXT EVENTカードの日程・会場を更新
2. pages/event.html のNEXT EVENTセクションの日程・会場を更新
3. 告知画像を images/flyers/riotXX-flyer.jpg にコピー
4. pages/entry-riotXX.html を作成（前回のentry-riotファイルをコピー）
   - タイトル・日程・会場を更新
   - 持ち時間・出演費・募集開始日を更新
   - ステータスを「準備中」に設定
5. pages/event.html の募集ボタンのリンク先を新しいファイルに変更
6. news.json に開催決定のNEWSを追加
```

## 補足

- entry-riotXX.html の出演費・持ち時間等は毎回変わる可能性があるため、必ず確認
- 告知画像は圧縮してから配置（幅800px程度）
- 前回のentry-riotファイルのフォームURL・スプレッドシートURLはコメントアウトまたは新しいURLに差替
