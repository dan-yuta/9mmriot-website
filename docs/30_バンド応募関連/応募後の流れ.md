## 応募方法の詳細

### 方法1: Google フォームから応募

```
応募者
  │
  ├── フォームにアクセス（HPのエントリーページから）
  │
  ├── 入力項目
  │   ├── バンド名（仮でもOK）
  │   ├── メンバー人数
  │   ├── 各メンバーのパートと名前（最大5人）
  │   ├── 代表者のメールアドレス
  │   └── 備考（6人以上の場合はここに追記）
  │
  └── 送信 → 主催がスプレッドシートに手動転記
```

### 方法2: XのDMから応募

```
応募者
  │
  ├── @9mmRIOT のDMに送信
  │
  ├── 記載内容
  │   ├── バンド名
  │   ├── 各メンバーのパートと名前
  │   └── 代表者のメールアドレス
  │
  └── 主催がスプレッドシートに手動記録
```

## 主催側の作業フロー

### 応募受付時

```
1. 応募を確認
   ├── フォーム回答一覧を確認
   └── X DMを確認

2. スプレッドシートに手動転記
   ├── 「バンドメンバープロフィール」シートにバンド名・パート・名前を記入
   ├── バンド名セルを青色に設定
   └── バンド代表者の名前セルを赤色に設定

3. 応募者に返信
   ├── メールまたはX DMで応募受付の確認
   ├── LINEグループの招待リンクを送付
   └── 主催からの返信をもってエントリー完了

4. LINEグループに参加を確認
```

### セトリ募集時（6月上旬頃）

```
1. LINEグループにてセトリ提出の案内を送信
2. セトリを回収・調整（曲被り回避）
3. 調整結果をLINEグループで共有
```

### イベント直前

```
1. タイムテーブルをLINEグループで共有
2. 当日の諸連絡（集合時間、持ち物、会場アクセス等）
3. 出演費の案内
```

## 連絡先の管理

| 連絡手段 | 用途 | タイミング |
|---|---|---|
| Google フォーム | 応募受付 | 募集期間中 |
| X DM | 応募受付（フォーム以外の手段） | 募集期間中 |
| メール | 応募確認・LINEグループ招待 | 応募受付直後 |
| **LINEグループ** | **以降の全連絡** | **応募後〜イベント当日** |

---

## 注意事項

- LINEグループには**バンドのリーダー（代表者）のみ**が参加
- リーダーが自分のバンドメンバーに連絡を伝達する
- 主催からの連絡はLINEグループに一本化する（メール・DMでの個別連絡は行わない）
- LINEグループは主催が作成・管理する
