# 出演バンド発表プロンプト

## 用途

出演バンドを発表する（EVENT詳細更新 + NEWS追加 + git push）。

## プロンプト

```
9mmRIOT vol.XX の出演バンドが決定しました。
以下の情報をもとにHPを更新してgit pushしてください。

【出演バンド（出演順）】
1. バンド名1
2. バンド名2
3. バンド名3
...

【作業内容】
1. pages/event.html のNEXT EVENTセクションの出演欄を更新
   - 「コピーバンド募集予定」→ 出演バンド名を記載

2. index.html のNEXT EVENTカードの出演欄も同様に更新

3. news.json に出演バンド発表のNEWSを追加
   - 日付: YYYY.MM.DD
   - カテゴリ: EVENT
   - タイトル: 9mmRIOT vol.XX 出演バンド発表
   - リンク先: pages/event.html
```
