# NEWS 更新手順

## 概要

NEWSの追加・編集は `news.json` ファイル1つを編集するだけで完了します。
トップページ（最新3件）とNEWS一覧ページ（全件）に自動で反映されます。

---

## 手順

### 1. news.json を開く

場所: `10_ソースコード/news.json`

### 2. 先頭に新しい記事を追加する

```json
[
  {
    "date": "2026.04.19",
    "tag": "ENTRY",
    "text": "9mmRIOT vol.09 出演バンド募集開始！",
    "link": "pages/entry-riot9.html"
  },
  {
    "date": "2026.04.09",
    "tag": "EVENT",
    "text": "9mmRIOT vol.09 開催決定！",
    "link": "pages/event.html"
  }
]
```

**ポイント: 新しい記事は配列の一番上に追加してください（上が最新）。**

### 3. 保存して完了

ブラウザをリロードすれば反映されます。

---

## 各項目の説明

| 項目 | 説明 | 例 |
|------|------|-----|
| `date` | 日付（`YYYY.MM.DD` 形式） | `"2026.04.19"` |
| `tag` | カテゴリ（下記4種類から選択） | `"EVENT"` |
| `text` | 本文（1行のテキスト） | `"9mmRIOT vol.09 開催決定！"` |
| `link` | リンク先（不要なら `""` 空文字） | `"pages/event.html"` |

## tag の種類

| tag | 表示 | 用途 |
|-----|------|------|
| `EVENT` | 赤ラベル | イベント開催・告知 |
| `INFO` | グレーラベル | サイト更新・一般お知らせ |
| `ENTRY` | 緑ラベル | 出演バンド募集 |
| `MEDIA` | 紫ラベル | 動画公開・メディア掲載 |

## link の書き方

| パターン | 書き方 |
|----------|--------|
| リンクなし | `"link": ""` |
| サイト内ページ | `"link": "pages/event.html"` |
| 外部サイト | `"link": "https://www.youtube.com/..."` |

※外部サイトのURLは自動で新しいタブで開きます。

## よくある例

### イベント告知

```json
{
  "date": "2026.06.01",
  "tag": "EVENT",
  "text": "9mmRIOT vol.09 タイムテーブル公開",
  "link": "pages/event.html"
}
```

### 出演バンド募集

```json
{
  "date": "2026.04.19",
  "tag": "ENTRY",
  "text": "9mmRIOT vol.09 出演バンド募集開始！",
  "link": "pages/entry-riot9.html"
}
```

### YouTube動画公開

```json
{
  "date": "2026.10.01",
  "tag": "MEDIA",
  "text": "9mmRIOT vol.09 ライブ配信アーカイブを公開しました",
  "link": "https://www.youtube.com/watch?v=XXXXXXXXX"
}
```

## 注意事項

- JSONファイルは **カンマの付け忘れ・余分なカンマ** でエラーになりやすいです
- 最後の記事の `}` の後にはカンマを付けないでください
- ダブルクォーテーション `"` を使ってください（シングルクォーテーション `'` は不可）
- ローカル（file://）で確認する場合、ブラウザのセキュリティ制限でJSONが読み込めないことがあります。その場合はローカルサーバーを使ってください

### ローカルサーバーの起動方法（簡易）

`10_ソースコード` フォルダで以下を実行:

```bash
# Python がインストールされている場合
python -m http.server 8000

# Node.js がインストールされている場合
npx serve
```

ブラウザで `http://localhost:8000` を開けば確認できます。
