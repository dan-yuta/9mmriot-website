# イベント開催後の更新プロンプト

## 用途

イベント開催後に、詳細ページ作成・過去イベント追加・YouTubeアーカイブ設定を一括で行う。

## プロンプト

```
9mmRIOT vol.XX が開催されました。以下の情報をもとにHP を更新してgit pushしてください。

【イベント情報】
- 日程: YYYY年M月D日（曜日）
- 会場: 会場名
- 開場/開演: OPEN HH:MM / START HH:MM
- チケット: ¥X,XXX（+1DRINK ¥XXX）

【出演バンド（出演順）】
1. バンド名1
2. バンド名2
3. バンド名3
...

【YouTubeアーカイブURL】
https://www.youtube.com/watch?v=XXXXXXXXXXX

【フライヤー画像】
（フライヤー画像のファイルパスを指定）

【作業内容】
1. pages/riotXX.html を作成（前回のriotファイルをコピーして内容を差替）
2. images/flyers/riotXX-flyer.jpg にフライヤー画像を配置
3. pages/event.html のPAST EVENTSにriotXXを追加
4. index.html の過去イベントカードを更新
5. index.html のMOVIEセクションのYouTubeサムネイルを最新に差替
6. pages/about.html のHISTORYを更新
7. news.json にアーカイブ公開のNEWSを追加
```

## 補足

- YouTubeサムネイルは `https://img.youtube.com/vi/動画ID/maxresdefault.jpg` で自動取得
- フライヤー画像は事前に圧縮（幅800px、quality 75）を推奨
- 前後ページのナビゲーション（前へ/次へ）のリンクも更新が必要
