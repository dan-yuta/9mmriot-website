# 全体レビュー依頼プロンプト

## 用途

定期的にサイト全体の品質をチェックする。公開前やイベント前に実施推奨。

## プロンプト

```
D:\200_OneDrive\OneDrive\10_9mmRIOT関連\999_HP関連\10_ソースコード
の9mmRIOT公式サイトを全体レビューしてください。

【チェック観点】
- リンク切れがないか
- スマホ表示で崩れていないか
- テキストの誤字・情報の古さがないか
- 「。」で改行されていない箇所がないか
- NEWSの日付・内容が正しいか
- NEXT EVENTの情報が最新か
- フッターのコピーライト年が正しいか
- ABOUTの統計数字（開催回数・バンド数・年の歴史）が正しいか

問題があれば修正してgit pushしてください。
```
