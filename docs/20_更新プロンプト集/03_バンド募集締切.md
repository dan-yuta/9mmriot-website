# バンド募集締切プロンプト

## 用途

バンド募集を締め切る（エントリーページの更新 + NEWS追加 + git push）。

## プロンプト

```
9mmRIOT vol.XX のバンド募集を締め切ります。
以下の対応をしてgit pushしてください。

1. pages/entry-riotXX.html のエントリーセクションを更新
   - フォームボタンを削除またはコメントアウト
   - 「募集は締め切りました」のテキストに変更

2. news.json に募集締切のNEWSを追加
   - 日付: YYYY.MM.DD
   - カテゴリ: ENTRY
   - タイトル: 9mmRIOT vol.XX 出演バンド募集を締め切りました
   - リンク先: なし
```

## 補足

- Googleフォーム側でも「回答を受付中」をOFFにする
